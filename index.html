JS函数的执行时机

1 解释为什么如下代码会打印 6 个 6 let i = 0; //声明变量i=0 for (i = 0; i < 6; i++) { setTimeout(() => { console.log(i); }, 0); //令i=0,1,2,3,4,5循环六次,每次循环结束，i=6 } 如果没有setTimeout，每次判断完i<6后，直接打印当前的 i，也就是 i=0,1,2,3,4,5; 即打印 i 在循环结束前。 因为有setTimeout，指过一会儿后，尽快打印出 i，因此每次循环结束后才会打印 i，而每次循环结束后 i=6。所以会打印 6 个 6。 2 写出让上面代码打印 0、1、2、3、4、5 的方法 for (let i = 0; i < 6; i++) { setTimeout(() => { console.log(i); }, 0); }
