1.a标签的用法

  • 跳转外部页面
  • 跳转内部锚点
  • 跳转到邮箱或电话

首先输入以下命令，安装http-server或parcel，用来访问本地网页。

yarn global add http-server
yarn global add parcel

安装后，用命令

http-server . -c-1
http-server -c-1
hs -c-1
或
parcel 文件名.html

来访问本地网页，不要双击HTML来访问自己本地的网页。

a标签的href属性

网址
<a href="//xiongzhongke.xyz" target="_blank">超链接</a>
<a href="https://xiongzhongke.xyz" target="_blank">超链接</a>
<a href="http://xiongzhongke.xyz" target="_blank">超链接</a>

路径
<a href="/a/b/c.html">我是C.html</a>
<a href="./index.html">我是index.html</a>

伪协议
<a href="javascript:;">我要一个a标签，但什么也不做，不刷新也不置顶</a>
<a href="#xzk">自动跳转到"id=xzk"的p标签</a>
<a href="mailto:775126135@qq.com">发邮件给xzk</a>
<a href="tel:15072026695">打电话给我</a>

a标签的target属性

有_blank/_top/_parent/_self四个值

<a href="http://xiongzhongke.xyz" target="_blank">在空白页面打开指定网址</a>

<body>
   <a href="http://xiongzhongke.xyz" target="_top">有多个网页层层引用，在最顶层的网页打开指定网址</a>
   <div>
    <iframe src="我是引用的网页.html">
    </iframe>
   </div>
</body>

<a href="http://xiongzhongke.xyz" target="_parent">有多个网页层层引用，在本网页的上一层网页打开指定网址</a>
<a href="http://xiongzhongke.xyz" target="_self">在本页面打开指定网址</a>

其他值
<a href="http://xiongzhongke.xyz" target="xzk">重复利用同一个窗口打开不同的页面</a>
<a href="http://google.com" target="xzk">重复利用同一个窗口打开不同的页面</a>
可打开控制台，用"window.name"来验证。

古德拜项目
<body>
   <a href="https://google.com" target="xzk">google</a>
   <a href="https://baidu.com" target="xzk">baidu</a>
   
   <hr />
    <iframe 
    style="border: none; width: 100%; height: 800px;"
    src=""
    name="xzk">
    </iframe>   
</body>

href的作用是，当用户点击“超链接”，可以打开指定网页"xiongzhongke.xyz”。 target的作用是，在指定页面打开超链接指定的网页。

iframe标签的作用是内嵌窗口，现在已经很少使用了，不要过多研究它。

2.img标签的用法

发出get请求，展示一张图片。

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta http-equiv="X-UA-Compatible" content="ie=edge" />
  <title>Document</title>
  <style>
    * {
      margin: 0;
      padding: 0;
      box-sizing: border-box;
    }
    img {
      max-width: 100%;
    }
  </style>

style里面的内容，使图片在手机端浏览时，与手机屏幕等宽。

</head>

<body>
  <img id="xzk" width="400" src="dog.png" alt="一只狗子" />
  <script>
    xzk.onload = function () {
      console.log("图片加载成功")
    }
    xzk.onerror = function () {
      console.log("图片加载失败")
      xzk.src = "/404.png"
    }
  </script>
</body>

script里的代码使得可以用控制台的console查看图片加载成功和失败的情况。 alt属性是当图片加载失败时，能看到对图片的文字说明。 src属性是图片来源。 width是图片宽度。

3.table标签的用法

table标签用来做表格。

  • 相关标签 table/thead/tbody/tfoot/tr/td/th
  • 相关样式

table-layout,控制表格和列的宽度。

table-layout: auto; auto大多数浏览器采用自动表格布局算法对表格布局。表格及单元格的宽度取决于其包含的内容。

table-layout:fixed; fixed表格和列的宽度通过表格的宽度来设置，某一列的宽度仅由该列首行的单元格决定。

border-collapse,控制单元格取消间距。

border-spacing,控制单元格间距。

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <meta http-equiv="X-UA-Compatible" content="ie=edge" />
  <title>Document</title>
  <style>
      table {
      width: 600px;
      table-layout: auto; 
      border-collapse: collapse;
      border-spacing: 0px;
    }
      td
      th {
      border: 1px;
    }
  </style>
</head>

<body>
    <table>
    <thead>
      <tr>
        <th>表头</th> //table head//
      </tr> //table row//
    </thead>
    <tbody>
      <tr>
        <th>表头</th>  
        <td>数据1</td> //table data//
        <td>数据2</td>
        <td>数据3</td>
      </tr>
    </tbody>
    <tfoot>
      <tr>
        <th>表头</th>  
        <td>数据1</td> //table data//
        <td>数据2</td>
        <td>数据3</td>
      </tr>
    </tfoot>
  </table>
</body>

</html>

4.form标签的用法

<form>元素表示了文档中的一个区域，此区域包含有交互控制元件，用来向 Web 服务器提交信息。

  • action属性：一个处理此表单信息的程序所在的URL。
  • method属性：浏览器使用这种HTTP方式来提交表单。可能的值有post和get
  • target属性：用来指示在提交表单之后，在哪个网址显示收到的回复。
  • autocomplete属性：on：浏览器能够根据用户之前在表单里输入的值自动补全。（input记得加入 name="username”）off：浏览器不会自动补全输入。
<body>
  <form action="/xzk" method="POST" autocomplete="off" target="_blank">
    <input name="username" type="text" />

    <button type="submit">
      <strong>
        xzk
      </strong>
      <img src="dog.png" alt="一只狗子">
    </button>
    <textarea style="resize: none; width: 50%;height: 300px" > </textarea>
  <hr />
  <select>
      <option value="">-请选择-</option>
      <option value="1">星期一</option>
      <option value="2">星期二</option>
    </select>
  </form>
</body>

input标签的各项属性可以用来制作单选(type="radio” name="gender”)，多选（type="checkbox” name="hobby”），提交文件(type="file”)。

textarea标签用来提交一段文字或文章。

select+option标签可以用来制作下拉选择按钮。
