1.HTML 是谁发明的

  • Tim Berners-Lee，1990年发明。Tim Berners-Lee，1976年毕业于牛津大学物理系，1984年进入欧洲原子核研究会（CERN）粒子实验室工作。父母都是数学家，参与过世界上第一台商业电脑，曼切斯特1型（Manchester Mark I）的建造。2004年，英女皇伊丽莎白二世向Tim Berners-Lee颁发大英帝国爵级司令勋章。
  • 李爵士发明html的初衷：为了使欧洲各国的核物理学家能通过计算机网络及时沟通传递信息进行合作研究。
  • 如今Web之父Tim Berners-Lee已经功成名就。在www发明之初，tim预见到一旦他的浏览/编辑器问世，势必引起网络软件大战，使国际互联网陷入割据分裂，为了他所钟爱的WWW事业，他决定在WWW的百家争鸣中扮演一个技术直辖市的角色，而不是角逐财富的商人。与那些依托互联网一夜暴富之士相比，蒂姆仍然坚守在学术研究岗位上，那种视富贵如浮云的胸襟，真正表现了一个献身科学的学者风度。

4.全局属性有哪些

  • class属性 用来对网页元素进行分类
<p class="xzkexgirlfriend">我的前女友：高圆圆</p>
<p>我最喜欢的演员：周星驰</p>
<p class="xzkexgirlfriend">我的前女友：新垣结衣</p>

上面代码中，第一个<p>和第三个<p>是一类，因为它们的class属性相同。

  • contenteditable属性 允许用户修改内容
<p contenteditable="ture">
鼠标点击，本句内容可以修改
</p>
  • hidden属性 是一个布尔属性，表示当前网页元素不再跟页面相关，因此浏览器不会渲染这个元素。
<p hidden>本句不会显示在页面上。</p>
  • id属性 是元素在网页内唯一的标识符，id属性值必须是全局唯一的，同一个页面不能有两个相同的id属性。
<p id="xiongzhongke1"></p>
<p id="xiongzhongke2"></p>
<p id="xiongzhongke3"></p>

id属性的值不能包含空格

  • style属性用来指定当前元素的CSS样式。具体的设置，请看CSS教程。
<p style="color: red;">hello xzk</p>

上面代码指定文字颜色为红色。

  • tabindex属性 方便用户使用Tab键遍历网页元素。按下Tab键，网页焦点从一个元素转移到另一个元素，tabindex属性的值是一个整数，表示用户按下Tab键时，网页焦点转移的顺序。
<p tabindex="1">这段文字可以获得焦点。</p>
  • title属性 用来为元素添加附加说明，鼠标悬浮在元素上面时，会将title属性值作为浮动提示，显示出来。
<div title="版权说明">
<p>本站内容使用创意共享许可证，可以自由使用。</p>
</div>

5.常用的内容标签有哪些，分别是什么意思（a、strong、em、code、pre 等等）

  • ol+li(ordered list+list item)
<ol>
  <li>列表项A</li>
  <li>列表项B</li>
  <li>列表项C</li>
</ol>
  • ul+li(unordered list +list item)
<ul>
  <li>列表项A</li>
  <li>列表项B</li>
  <li>列表项C</li>
</ul>
  • dl+dt+dd(description list+term+data)

<dl>标签是一个块级元素，表示一组术语的列表（description list）;术语名（description term）由<dt>标签定义，术语解释（description detail）由<dd>标签定义;<dl>常用来定义词汇表。

<dl>
  <dt>CPU</dt>
  <dd>中央处理器</dd>

  <dt>Memory</dt>
  <dd>内存</dd>

  <dt>Hard Disk</dt>
  <dd>硬盘</dd>
</dl>

上面代码的默认渲染结果如下:

CPU
  中央处理器

Memory
  内存

Hard Disk
  硬盘
  • pre(preview的缩写)

有保留空格的作用。

  • hr(水平分割线)
  • br(换行)
  • a(anchor的缩写)

链接通过<a>标签表示，用户点击后，浏览器会跳转到指定的网址。 标签内部不仅可以放置文字，也可以放置其他元素，比如段落、图像、多媒体等等。

<a href="https://xiongzhongke.xyz/">熊仲轲的网站</a>
<a href="https://www.example.com/">
  <img src="https://www.example.com/foo.jpg">
</a>
  • em(emphasis的缩写)

字体倾斜，表示语气强调。

  • strong

字体加粗，表示本质强调。

  • code

展示代码用。

  • q(quote的缩写)

表示引用。

  • blockquote

块级引用。
