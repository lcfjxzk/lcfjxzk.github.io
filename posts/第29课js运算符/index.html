  • 自增自减x++/++x/x–/–x
var a = 1
a++ //值为1，值为++之前的值
++a //值为2，值为++之后的值
++ //让a加1

a++/a–，a在前值为前； ++a/–a，a在后值为后

注意：永远不要用"=="要用"===”（可参见真值表）

“=="会自动类型转换，导致JS三位一体。 

[] !== []//空数组不全等于空数组
{} !== {}//空对象不全等于空对象
NaN !== NaN

二、防御性编程（可选链语法）/布尔运算符

  • 以防console不存在报错（console有可能被人改写）
console && console.log && console.log("hi")
//如果前两个都成立，我就执行第三个

以下为明年最新的防御性编程语法

console?.log?("hi")
//第一项成立继续往下走，第二项成立继续往下走。
  • n的保底值
function add(n){
    if(!n){n=0}else{n=n}
    n = n || 0//上一句的简写法，但是有bug，当n为字符串时不成立
return n+1
}

三、二进制运算符

  • 或、与、否

|、&、~

0b11 === 3//0b表示二进制数
(0b1111 | 0b1010).toString(2)
-->"1111"//toString(2)表示转化为二进制字符串。"|"表示或，重叠取值，1和0取1，1和1取1，0和0取0。只要有一个1，就重叠取1。
(0b1111 & 0b1010).toString(2)
-->"1010"//必须两个1才能取1，1和0取0。

关于二进制运算符，有一个补码的内容需要自学，如果在视频里讲，需要很长时间。

  • 异或

^

两个位相同，则结果为0；两个位不同，则结果为1

(0b1111 ^ 0b1010).toString(2)
"101"
  • 左移右移

“»”

将二进制数整体向左向右移动

(0b0010 >> 1).toString(2)//0001
-->"1"

  • 头部补零的右移运算符

“»>”

void的作用：如果是表达式就求值，如果是语句就执行语句。然后得到一个"undefined”. void的唯一目的就是得到一个"undefined”

<a href="https://baidu.com" onclick="console.log('hi')";return false>baidu</a>

<a href="javascript:;" onclick="console.log('hi')">baidu</a>

五、运算符优先级

JS优先级你这辈子也记不下来。遇到运算符优先级的题目，直接放弃。今天是一节特别丧的课。

mdn可查询 https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Operators/Operator_Precedence

技巧：

  • 圆括号优先级最高
  • 会用圆括号就行
  • 其他一律不记

面试官如果问你你为什么不记优先级，回答如下：

因为运算符优先级太多了，就算我记下来，我也不能确保我的同事能记清楚，所以我只记了优先级最高的圆括号，我会在写代码的时候非常清楚的表明我的优先级，这样我的同事就能很好的理解我的代码。 从让同事理解的角度来反驳面试官。

==为什么不学？

因为平时都用"===”，这样写代码更加清晰简洁。
