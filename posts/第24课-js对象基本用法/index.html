第24课 JS对象基本用法

January 11, 2020

一、对象的定义、声明对象的两种语法

  1. 对象的定义
  • 无序的数据集合
  • 键值对的集合
let obj = { name: "frank", age: 18 };
let obj = new Object({ name: "frank" });
console.log({ name: "frank", age: 18 });
  1. 对象的属性

键 key–>'name' 也叫对象的属性名。

键名是字符串。键名的引号可以省略，省略后只能写标识符，不能写汉字或表情符 emoji。

值 value–>'frank' 也叫对象的属性值。

键值对–>'name':'frank'

Object.keys(obj)可以得到 obj 的所有 key。

  1. 变量作属性名
let p1 ='name'
let obj ={p1:'frank'},属性名为字符串'p1'
let obj = {[p1]:'frank'},这样写,属性名为变量p1,变量求值后属性名为'name'
  1. 隐藏的属性名

JS 中每一个对象都有一个隐藏属性，隐藏属性存储着共有属性组成的对象的地址，这个共有属性组成的对象叫做原型。

二、如何删除对象的属性

delete obj.xxx;
或;
delete obj["xxx"];

以上代码删除 obj 的 xxx 属性名和属性值。

obj.name = undefined;

以上代码只删除属性值。

如何查看是否删除成功？输入以下代码

"xxx" in obj;

返回值为 false,表示删除成功，即判断该属性不在该对象里面；返回值为 true 表示没有删除。

三、如何查看对象的属性

Object.keys(obj)查看自身所有属性

console.dir(obj)查看自身+共有属性

obj.hasOwnProperty('toString')判断一个属性是自身的还是共有的 'toString in obj'查看是否有这个属性。

查看单个属性

优先使用obj['key']查看

对象的原型也有原型

四、如何修改或增加对象的属性

  1. 修改或增加对象的自身属性

直接赋值

let obj = { name: "frank" };
obj.name = "frank";
obj["name"] = "frank";
let key = "name";

批量赋值

Object.assign(obj, { age: 18, gender: man });
  1. 修改或增加对象的共有属性

不推荐用以下方法修改共有属性

推荐用以下方法修改共有属性

window.Object.prototype.toString = "yyy";

一般不要修改共有属性，会引起很多问题。

  1. 原型链

原型链图解

推荐用以下方法创建原型

var person = Object.create(common);
表示以common为原型来创建person;
let obj = Object.create(common);
表示以common为原型来创建obj;

不推荐用以下方法创建原型

person.__proto__ = common;
不推荐;

效能很低，所以不推荐。

五、‘name’ in obj 和 obj.hasOwnProperty(‘name’) 的区别

'name' in obj

  • 返回值为 true

‘name'是对象 obj 的一个属性，有可能是自身属性也有可能是共有属性。

  • 返回值为 false

‘name'不是对象 obj 的属性。

obj.hasOwnProperty('name')

  • 返回值为 true

    ‘name'是对象 obj 的自身属性。

  • 返回值为 false

    ‘name'不是对象 obj 的自身属性。
