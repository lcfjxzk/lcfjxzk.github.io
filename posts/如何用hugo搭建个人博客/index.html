如何用hugo搭建个人博客

December 8, 2019

  1. 安装hugo 在官网上下载，并把hugo.exe路径放到PATH中

    • brew install hugo
    • hugo version

  2. 进入hugo官网，从step2开始抄代码，到step7，得到public目录

    • hugo new site lcfjxzk.github.io-generator
    • git init
    • git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
    • echo ‘theme = “ananke”’ » config.toml
    • hugo new posts/开博大吉.md #编辑博客内容
    • hugo server -D #Navigate to your new site at http://localhost:1313/.
    • hugo -D #得到public目录

  3. 添加要写的博客，将内容部署到GitHub pages 上

    • cd public
    • git add .
    • git commit -v
    • git init
    • git remote add origin git@github.com:lcfjxzk/lcfjxzk.github.io.git
    • git push -u origin master

  4. 在github远程仓库settings中GitHub pages上点开链接即可。

  5. 如何在 Hugo 里插入图片

  • 将图片文件 1.png 存放到 static 目录里，得到 static/1.png
  • 在博客 Markdown 里写 ![](/1.png) 就能看到 static/1.png 对应的图片了
  • 如果你想在 static 里创建 images 目录，直接创建就好，对应的图片前面加上 /images 即可，比如：
  • 将图片文件 2.png 放到 static/images 目录里，得到 static/images/2.png
  • 在博客里写 ![](/images/2.png)即可

推荐使用 static/images 来存放图片，这样更有条理。 推荐使用 VSCode 的拖放功能存入图片。
